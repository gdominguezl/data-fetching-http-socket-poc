# Server

## Getting started
1. Go to server folder:
    ```
    cd server
    ```
2. Install all dependencies:
    ```
    pnpm install
    ```
3. Run Nest server:
    ```
    pnpm run dev:start
    ```

# Client

## Getting started
1. Go to client folder:
    ```
    cd client
    ```
2. Install all dependencies:
    ```
    npm install
    ```
3. Run React app:
    ```
    npm run start
    ```
